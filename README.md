# SalesPulse-Regional-Product-Performance-Dashboard
A Power BI dashboard built to analyze business health, track profitability, and uncover product and region-wise sales insights using an end-to-end ETL process.

## ğŸ“ Project Overview

Understanding sales performance at a regional and product level is crucial for strategic business planning. This dashboard provides a clear view of:

- Monthly sales trends and profitability
- Product category and customer segment performance
- Regional sales breakdown
- Top-performing items and geographies

Built in **Power BI**, the project uses a full ETL (Extract, Transform, Load) pipeline to connect, clean, relate, and visualize data effectively.
